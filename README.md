# AI-4-BC-Consultants
**Target audience**: Functional ERP Consultants. Microsoft D365 Business Central (BC) focus is assumed but general principles could be adopted elsewhere.

**Goal**: Provide prompts and sample files that could be used in the ERP project implementation lifecycle of BC, as follows.

**Reference materials for webinar**: AI-Enabled Delivery for Consultants - 2026-06-29 - https://areopa.academy/youtube/

## Scenario scope and AI benefits targeted

### Phase 1: Planning & Design
**AI benefit**: Accelerating Requirements Gathering
Activities:
- Extract requirements from workshop notes
- Clarify vague requirements
- Detect gaps and assumptions
- Check for functional completeness in the context of BC

**AI benefit**: Improving Analysis & Understanding
Activities:
- Process simplification
- Consistency analysis
- Risk-focused analysis
- Implicit requirement discovery

### Phase 2: Implementation

**AI benefit**: Faster & Better Solution Design
Activities:
- Challenging the design
- Alternative approaches
- Customisation pressure test
- Scenario simulation

**AI benefit**: Improving Breadth & Speed of Testing
Activities:
- Test case generation
- Negative testing
- Role-based testing
- Regression awareness

### Phase 3 Go-Live & Phase 4 Aftercare:

**AI benefit**: AI-Supported Documentation
Activities:
- User Guide creation
- Audience-specific rewriting
- Standardisation
- Living documentation

**AI benefit**: Assisting with Issue Triage and RCA
Activities:
- Summarise an issue
- Root cause hypothesis
- Change correlation
- Preventative actions
